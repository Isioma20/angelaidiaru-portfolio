# Idiaru Angela - Frontend Portfolio

A personal web portfolio built to showcase frontend engineering projects.

## Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** GSAP & ScrollTrigger
- **API Integration:** GitConnected API

## Local Setup

1. Clone the repository and navigate to the project directory:
   ```bash
   git clone https://github.com/Isioma20/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
